# Domain Setup (Vercel)

## Deploy to Vercel

1. Create a Vercel account
2. Import your GitHub project
3. Add all required Environment Variables
4. Deploy

## Add custom domain

1. In Vercel project, open **Settings**
2. Open **Domains**
3. Add your domain (example: `smartreplypro.com`)

## DNS setup

Configure DNS with your domain provider using either:

- **A record**, or
- **Nameservers** (recommended by Vercel flow)

Follow the exact DNS values shown by Vercel.

## SSL

Vercel enables SSL certificates automatically once DNS is configured correctly.
